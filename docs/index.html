CSharp-Web-API

Professional Web API by Gabriel Demetrios Lafis

Welcome to CSharp-Web-API!

This project showcases a professional Web API developed in C# and .NET. It's designed to demonstrate best practices in RESTful API development, including clean architecture, unit testing, and comprehensive documentation.

Explore the repository on GitHub for more details:

  • GitHub Repository
  • Gabriel Demetrios Lafis's GitHub Profile

Key Features:

  • Complete RESTful API with CRUD operations for product management.
  • Organized project structure following clean architecture principles.
  • Comprehensive unit tests ensuring code reliability.
  • Interactive API documentation using Swagger/OpenAPI.
  • Developed with .NET 8 for optimized performance.

API Documentation (Swagger UI):

Once the API is running locally, you can access the interactive Swagger UI at https://localhost:7000/swagger to explore and test the endpoints.

For detailed instructions on how to set up and run the API, please refer to the README.md file in the repository.
